1️⃣ What is the difference between var, let, and const?
    Ans: var - var is function-scoped. 
         let - let is block-scoped. Can change value.
         const - const is also block-scoped but cannot reassign value.


2️⃣ What is the spread operator (...)?
    Ans: The spread operator merge multiple array , copy arrays and pass value of function.


3️⃣ What is the difference between map(), filter(), and forEach()?
    Ans: map() - transform elements
         filter() - Select elements based on condition
         forEach() - loop through elements.
